DriveLog — Google Sheets Setup

Connect your tracker app to Google Sheets so data flows automatically

1 Create Your Google Sheet

  1. Go to sheets.google.com and create a new spreadsheet
  2. Rename it to DriveLog (or whatever you like)
  3. Create 3 sheets (tabs) with these exact names:
    • Milage Tracker
    • Reimbursement Tracker
    • Maintenance Tracker
  4. Add headers to each sheet:

Milage Tracker — Row 1:

Date | Odometer | Price/Gal | Spend | Volume | City | Gas Station | Miles Driven | Comments

Reimbursement Tracker — Row 1:

Date Submitted | Milage | Value | Date Received

Maintenance Tracker — Row 1:

Date | Service | Spend | Milage | Notes
Tip: You can copy your existing data from the Excel file into this Google Sheet to start with your current records.

2 Add the Apps Script

  1. In your Google Sheet, go to Extensions → Apps Script
  2. Delete any existing code in the editor
  3. Paste the following code:
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet();
  var data = JSON.parse(e.postData.contents);

  var sheetMap = {
    'fuel': 'Milage Tracker',
    'reimbursement': 'Reimbursement Tracker',
    'maintenance': 'Maintenance Tracker'
  };

  var targetSheet = sheet.getSheetByName(sheetMap[data.sheet]);
  if (!targetSheet) {
    return ContentService.createTextOutput(
      JSON.stringify({status: 'error', message: 'Sheet not found'})
    ).setMimeType(ContentService.MimeType.JSON);
  }

  targetSheet.appendRow(data.data);

  return ContentService.createTextOutput(
    JSON.stringify({status: 'success'})
  ).setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet();
  var result = {};

  var sheets = ['Milage Tracker', 'Reimbursement Tracker', 'Maintenance Tracker'];
  sheets.forEach(function(name) {
    var s = sheet.getSheetByName(name);
    if (s) {
      var data = s.getDataRange().getValues();
      result[name] = data;
    }
  });

  return ContentService.createTextOutput(
    JSON.stringify(result)
  ).setMimeType(ContentService.MimeType.JSON);
}

3 Deploy as Web App

  1. Click Deploy → New deployment
  2. Click the gear icon next to "Select type" and choose Web app
  3. Set these options:
    • Description: DriveLog
    • Execute as: Me
    • Who has access: Anyone
  4. Click Deploy
  5. Click Authorize access and go through the Google permissions flow
  6. Copy the Web app URL — it looks like:
    https://script.google.com/macros/s/AKfyc.../exec
Important: If Google shows a "This app isn't verified" warning, click Advanced → Go to DriveLog (unsafe). This is normal for personal scripts.

4 Connect the Tracker App

  1. Open your DriveLog app
  2. Tap the SETUP tab at the bottom
  3. Paste the Web app URL into the Sheets URL field
  4. Tap Save Settings
  5. Try logging a test entry — check your Google Sheet to confirm it appeared!

5 Connect the Dashboard (Optional)

Once your data lives in Google Sheets, your dashboard can pull from it live. Just ask Claude to rebuild the dashboard to read from your Google Sheet URL, and it will auto-refresh whenever you reload the page.

Tip: Bookmark the tracker app on your phone's home screen for quick access. In Safari, tap the share icon → "Add to Home Screen". In Chrome, tap the menu → "Add to Home screen".
